Парсер новостей с портала Lenta.ru.

Написан на PHP с использованием архитектуры MVC и BOOTSTRAP.

Для запуска необходимо:

- импортировать себе базу данных "phpparserlenta.sql";

- в файле components => DB.php нужно поменять настройки под свой хость и sql;

- на сервере должен быть установлен Rewrite-module;

- для работы использоуется Composer, поэтому в командной строке необходимо выполнить composer install.

На сайте реализовано:

- список всех полученных новостей;

- просмотр полной информации новости;

- при нажатии на кнопку "Получить последние новости" в список ниже добавляются свежие материалы;

- при нажатии на кнопку "Экспорт" генерируется CSV файл новостей за последние сутки;
